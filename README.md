## Local development (permanent command)

Run this once:

```bash
chmod +x bin/dev
```

Then always start local live server with:

```bash
./bin/dev
```

This command:
- installs gems if needed,
- stops stale Jekyll serve processes,
- runs Jekyll with LiveReload and your dev config.
